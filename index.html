What is HTML?

HTML stands for Hyper Text Markup Language. HTML is not a programming language. It is the skeleton or structure of a web page. And HTML tells the browser what to display in web page. HTML is consist of series of elements. If you want to be a web developer, this language is the first one you will learn. This language is easy to learn and it will take you 1 -2 weeks to master it if you are focus.

HTML Tag's Anatomy

All text and other elemens that you can see on the webpage is enclosing with tags. HTML is consist of many tags. There are two type of tags: Opening tag(<>) and Closing tag(</>). And the elements that inside the opening and closing tag is called content.

<p>I am a paragraph.</p>

Explanation:

  • In the example above, the opening tag is <p> which tells the browser that it is a paragraph.
  • While the closing tag is the <\p> which tells the browser the it is the end of the paragraph.
  • The content is the text inside the tag. In the example, the 'I am a paragraph' is the content of p tag.

Note: If you start with p tag(<p>), the closing tag should be the same but there's a slash before the p.

Not all element in HTML have closing tag. Ther'es an element that have no closing tag like <img>, <link>, <input>.

Head and body Tag

Head tag (<head>) and Body tag (<body>) is one of the most important tag in HTML. All Web Page should have head and body tag to function properly.

Inside the Head section (<head>):

  • Contains metadata which means that it contains information about your web page like author of the page..
  • Contains link of your CSS, the appearances of your web page, and link of your Javascript, the functions of your web page.
  • Example: <metadata>, <link>, <title>, etc.

      • Inside the Body section (<body>):

      • Contains elements that you can see on the web page such as heading and image.
      • Examples: <h1>, <p>, <a>, <img>, etc.
Headings

HTML headings can be define as a title or subtitle. There are 6 types of headings: <h1>, <h2>, ... <h6>. <h1> define as the most important text while the <h6> is the least. <h1> is commonly used in a home page as a main title.

<h1> I am the largest.<\h1>

<h2> I am larger.<\h2>

<h3> I am larger.<\h3>

<h4> I am larger.<\h4>

<h5> I am large.<\h5>

<h6> I am the smallest.<\h6>

Links

Do you want to jump or open another site by clicking the text or image? If yes, Anchor tag (<a>) can help you with that. Anchor tag can help you to jump to another site as well as to the specific section of your website.

<a href="www.google.com" target="_blank">Click me</a>

<a href="#web-section-id">Jump to page section</a>

In the above example,

  • href value means that what website you want open or what section do you want to jump.
  • target value means that if you want to open a new tab to open the link or not. The '_blank' means that I want to open a new tab. If you don't want to open a new tab and you want to stay on the same tab, you can remove the target attribute.

Not only text you can add a link, you can also add on image by doing this:

<a href="www.google.com" target="_blank"><img src="example.jpg" target="_blank"></a>

By clicking on the image, you will direct to google.com in a new tab.

List

HTML list can help you to group all related items. There are two types of list, the unorder list and order list. Unorder list commonly use in creating a navigation while the order list is usually use in creating a steps for something like steps in cooking of something.

The unorder list looks something like this:

  • Some text
  • Some text
  • Some text

<ul>

<li>Some text</li>

<li>Some text</li>

<li>Some text</li>

</ul>

Unorder list start with <ul> tag and the list items start with <li> tag

The order list looks something like this:

  1. Some text
  2. Some text
  3. Some text

<ol>

<li>Some text</li>

<li>Some text</li>

<li>Some text</li>

</ol>

Order list start with <ol> tag and the list items start with <li> tag, same as the unorder list.
